Java启动线程的方式

继承Thread

一个类继承 Thread，new出这个对象，然后调用 start 方法

实现 Runnable 接口

  • 一个类实现 Runnable 接口

  • 然后 new Thread 里面传进去他的对象

  • 调用线程对象的 start 函数

使用匿名内部类配合 lambda 表达式更简洁

实现 Callable 接口

  • Thread只能传入 Runnable 接口，但是没有返回值和异常。

  • 使用 FutureTask 可以让 Callable 变成 Runnable

  • 同样方式 new 线程，start

  • 再通过 get 得到返回值

（类FutureTask 实现了 接口 RunnableFuture，同时接口 RunnableFuture 继承了接口 Runnable）

注意 get 方法：

  • 结果会被缓存

  • 会阻塞等待

线程池

简要讲

  • 使用 Executor 工具类或者直接 new ThreadPoolExecutor 对象得到一个线程池（生产环境更推荐后者）

  • 调用线程池方法 execute 或者 submit

  • 最后销毁线程池
