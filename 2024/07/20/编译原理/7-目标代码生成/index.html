目标代码生成

目标代码生成

IR -> 目标代码（二进制）机器码/汇编代码

汇编代码（Assembly） 和 汇编程序（Assembler）

  • 汇编代码：一种接近于机器码的编程语言

  • 汇编程序：将汇编代码转化为实际的机器码（01串），偶尔会把这部分工作给 Linker 来做

链接器（Linker）

汇编程序将源码转为机器码，但是存在缺漏（各种引用外部库），并且单独编译每个文件，生成的是 object file（可重定向文件 relocatable object file ）没办法直接运行。这时候需要链接器（Linker） 将源码和库填补到正确的地址，产生一个可执行文件。
