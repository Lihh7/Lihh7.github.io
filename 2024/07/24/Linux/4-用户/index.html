Linux用户

登录尽量少用 root 账号，拥有最大权限

  • 登录需要权限，使用 su - root 切换为 root

  • logout 注销用户，在运行级别3，无图形有效；退出普通的用户会退出连接

linux 系统是多用户系统，只能有一个 root ，可以添加其他用户

  • 添加用户 useradd 用户名 -m ，其中 -m 表示添加家目录

  • 也可以指定目录添加用户 useradd -d /home/test king -m 即指定用户目录

  • 设置密码 passwd 用户名 默认当前用户的密码

  • 删除用户保留家目录 userdel milan ，需要 root 权限，不要自己删自己

  • 删除用户及家目录 userdel -r tom ，一般删除需要保留家目录

  • 查询用户信息 id 用户名

  • 切换用户 su - 用户名，高权限切换到低权限不需要密码，反之需要；返回则 logout/exit

  • who am i 查询第一次登录的信息

pwd 可以显示当前所在目录

shell 有 bash ，fcsh ，csh 可以把命令解释给内核

用户组：对共性（权限）的用户组成一组

  • 新增组 groupadd wudang(组名)

  • 删除组 groupdel wudang(组名)

  • 添加用户到某个组 useradd -g wudang zwj ，如果不指定组则默认添加到自己同名的一个组

  • 修改用户到某个组 usermod -g mojiao zwj
