Linux常用指令

在 linux 下以 . 开头的是隐藏文件

选项可以组合使用 -al 或者 -a -l

写绝对路径前面不用 /

帮助

  • man 获得帮助信息，基本语法 man [命令或者配置文件]

  • help 命令，获得 shell 内置的命令的帮助信息

文件目录类

  • pwd 指令，显示当前工作目录的绝对路径

  • cd 指令，可以定位绝对路径和相对路径

  • cd ~ 回到家目录

  • cd .. 回到上一级目录

  • ls 显示当前目录下的文件和文件夹，-a 包括隐藏，-l 单列显示

  • mkdir 创建目录，-p 创建多级目录

  • rmdir 删除空目录，有内容无法删除

  • rm -rf 删除非空目录，这里其实用了 -r 递归和 -f 不提醒

  • touch 创建一个空文件

  • cp 拷贝文件到指定目录，cp [选项] source sest 常用选项 -r 递归复制整个文件夹，可以用 \cp 强制覆盖不提醒，比如

  • rm 删除文件或者目录，-r 递归删除，-f 强制删除不提醒

  • mv 移动文件或者重命名，格式 mv old new ：如果在同一个目录则是重命名，如果不在则移动，也可以移动并重命名；也可以移动目录

  • cat 查看文件内容，-n 显示行号，更安全不会修改，一般加上管道命令（前面结果交给后面） | more

  • more 指令，基于 vi 编辑器文本过滤器，也可以直接用 more 来查看

  • less 指令分屏查看文件内容，根据显示加载内容，不会一下子加载，效率较好

  • echo 输出内容到控制台

  • head 输出前 10 行，也可以看前 5 行 head -n 5 ...

  • tail 显示后面 10 行，同样可以指定行数 -n 5，还可以用来实时监听文本内容

  • > 输出重定向把输出写入（覆盖）到其他地方，比如 echo "hello" > /home/mydate.txt

  • >> 输出重定向把输出写入（追加）到其他地方

  • ln 软链接（符号链接），类似于快捷方式，存放了链接其他文件的路径，基本语法 ln -s [源文件或者目录] [软连接名] 如果 cd 进去会跑到链接的地方
  • history 查看已经执行过的命令

date 指令-显示当前日期

  • date 显示当前时间

  • date +%Y 显示当前年份

  • date +%m 显示当前月份

  • date +%d 显示天

  • date "+%Y-%m-%d %H:%M:%S" 格式化输出

也可以设置系统时间

  • date -s "2020-11-03 20:02:10"

查看日历

  • cal

  • cal 2020

搜索查找类

  • find [搜索范围] [选项] 往下递归搜索，选项 -name ，-user，-size +200M(k,M,G)

  • locate 快速定位文件路径，事先建立数据库无需遍历整个文件系统。需要首先 updatedb

  • which 可以查看某个指令在哪个目录

  • grep 指令过滤查找。-n 显示匹配行及行号，-i 忽略大小写。

  • | 管道符号，将前一个命令的处理结果传递给后面的命令。

压缩和解压

  • gzip/gunzip 指令，压缩和解压，文件名为 .gz

  • zip/unzip 压缩解压，-r 递归压缩，-d<目录> 指定目录

  • tar 指令，打包指令，最后打包后的文件是 .tar.gz 文件，可以打包多个
