线程相关方法

方法 static 功能
start() 启动一个新线程，在新的线程运行 run 方法
run() 新线程启动后会调用的方法，可以覆盖如果不覆盖传递Runnable则会调用接口的 run 方法
join() 等待线程运行结束
join(long n)
getId()
getName()
setName(String)
getPriority()
setPriority(int)
getState() 获取线程状态。分别为：
NEW, RUNNABLE, BLOCKED, WAITING,TIMED_WAITING, TERMINATED
isInterrupted() 判断是否被打断，不会清除打断标记
isAlive()
interrupt() 如果被打断线程正在 sleep，wait，join 会导致被打断的线程抛出 InterruptedException，并清楚打断标记；如果打断的正在运行的线程，则会设置打断标记 ；park 的线程被打断，也会设置打断标记
interrupted() static 会清除打断标记
currentThread() static
sleep(long n) static 让当前执行的线程休眠n毫秒，休眠时让出cpu的时间片给其它线程
yield() static 提示线程调度器让出当前线程对CPU的使用
